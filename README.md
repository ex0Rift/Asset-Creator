# Asset creator!

A small pixel based paint program written in C by myslef for [HackClub Midnight](https://midnight.hackclub.com "@HackClub"). it has basic functionality, RGB colour options
and option for drawing a rectangle, drawing, earsing and making selecions for moving.

if you would like to compile the source code yourself use the build.sh file.

The file saving dialogue uses [tinyfiledialogs v3.21.2 ](https://sourceforge.net/projects/tinyfiledialogs/ "@tineFileDialouge") all credit to them for the saving intergration.

## How to use

### Changing colour

  using the RGB values:
>select what youd like the red to be
>
>Select what you would like Green to be
>
>Select what you would like Blue to be

  Using Colour picker:

>Press the colour preview next to the RGB sliders
>
>A border will show around it to show it is "Selected
>
>bring your mouse to hover over the selected colour on the screen
>
>press the "ESC" key to stopp choosing colour and it will save.


### Drawing a Rectangle

> Select the "Cube" button
>
> Drag on the canvas to make the rectangle space
>
> Release mouse and rectangle will apear in selection of the colour shown in the colour preview

### Drawing / Erasing

>Select the tool you would like respectable to what is labled
>
>drawing tool will use your colour which is visible in the colour preview
>
>erase tool erases
>
>next to these two buttons you can change the brush size
>
>press + to increase brush size
>
>press - to decrease brush size


### Resizing canvas

>CLick "NEW" on the top bar
>
>two options are available
>
>top is the width
>
>bottom is height
>
>adjust them with the + and - buttons
>
>press create
